# 百度搜索资源平台提交指南

## 步骤1：登录百度搜索资源平台

1. 打开：https://ziyuan.baidu.com
2. 使用百度账号登录

## 步骤2：添加网站

1. 点击「站点管理」→「添加网站」
2. 输入：`baibiandance01.netlify.app` 或 `www.baibiandance.com`
3. 验证方式：选择「文件验证」或「HTML标签验证」

## 步骤3：验证网站

**文件验证（推荐）：**
1. 下载验证文件（baidu_verify_file.html）
2. 放到网站根目录：public/ 文件夹
3. 验证通过后删除

**HTML标签验证：**
在 index.astro 的 `<head>` 中添加：
```html
<meta name="baidu-site-verification" content="代码" />
```

## 步骤4：提交sitemap

1. 「普通收录」→「sitemap」
2. 提交：`https://baibiandance01.netlify.app/sitemap.xml`

## 步骤5：主动推送（可选）

获取推送接口地址，提交实时URL
