# 抖音/快手短视频SEO优化方案

## 为什么做短视频SEO？

- 抖音/快手是国内最大流量入口
- 短视频搜索权重越来越高
- 家长群体活跃度高

## 短视频内容脚本

### 脚本1：机构介绍（15-30秒）

```
【开头】
画面：机构环境，老师跳舞
配音：临汾家长给孩子选街舞培训，看完这视频再决定！

【主体】
画面：师资展示、资质证书、教学环境
配音：选机构看这5点：1师资专业 2资质齐全 3课程体系 4环境安全 5能试课

【结尾】
画面：联系方式
配音：百变街舞，8年教学经验，中国舞协会员单位，免费试课19035772680

#少儿街舞 #街舞培训 #临汾
```

### 脚本2：家长必看（15-30秒）

```
【开头】
画面：孩子跳舞开心画面
配音：给孩子报街舞班前，这5件事必须知道！

【主体】
逐条讲解5个要点（配合字幕）
1. 看资质
2. 看师资
3. 看课程
4. 看环境
5. 试课体验

【结尾】
配音：关注我，下期讲街舞考级有用吗

#少儿街舞 #家长必看 #街舞培训
```

### 脚本3：街舞知识科普（15-30秒）

```
【开头】
画面：hiphop vs jazz 对比视频
配音：Hiphop和Jazz有什么区别？初学者应该学哪个？

【主体】
- Hiphop：自由、节奏感、表达个性
- Jazz：优雅、节奏鲜明、舞台表现力

【结尾】
配音：根据孩子性格选择，适合最重要

#街舞 #Hiphop #Jazz #舞蹈科普
```

## 发布策略

1. **标题**：包含关键词+吸引眼球
2. **话题**：添加相关话题标签
3. **位置**：定位机构地址
4. **时间**：中午12-13点 / 晚上19-21点
5. **频率**：每天1-2条，持续更新
