# 百变街舞官网

临汾专业少儿街舞培训机构官网

## 技术栈
- Astro
- 静态网站部署到 Netlify

## 开发

```bash
# 安装依赖
npm install

# 本地开发
npm run dev

# 构建生产版本
npm run build
```

## 部署

推送到 GitHub 后，在 Netlify 添加这个仓库即可自动部署。

## 优化特性
- LocalBusiness Schema 结构化数据
- Organization Schema 组织信息
- FAQ Schema (25个常见问题)
- Course Schema (4大课程体系)
- Sitemap.xml 站点地图
- robots.txt 爬虫协议
- 百度SEO优化
